# NovoUso Amazônia

Projeto front-end estático desenvolvido em HTML, CSS e JavaScript.

O objetivo da aplicação é simular uma plataforma de reutilização, doação, venda e lance de itens, com páginas para entrada, cadastro, home, detalhes de item, histórico e outras telas relacionadas.

## Como executar o projeto

Este projeto deve ser executado no navegador usando a extensão **Live Server** no VS Code.

### Pré-requisitos

- Ter o [Visual Studio Code](https://code.visualstudio.com/) instalado
- Ter a extensão **Live Server** instalada no VS Code

### Passo a passo

1. Abra a pasta do projeto no VS Code.
2. Abra o arquivo `src/index.html`.
3. Clique com o botão direito sobre o arquivo.
4. Selecione **Open with Live Server**.
5. O navegador será aberto automaticamente com o projeto em execução.

## Estrutura do projeto

- `src/index.html` - página inicial do projeto
- `src/pages/` - páginas internas da aplicação
- `src/styles/` - estilos globais, layouts e componentes
- `src/assets/` - imagens, ícones e fontes

## Observações

- O projeto usa caminhos absolutos a partir de `/src`, então o ideal é rodar com o Live Server para evitar problemas de carregamento de arquivos.
- Se o navegador abrir uma página em branco, confira se você iniciou o Live Server a partir do arquivo `src/index.html`.
